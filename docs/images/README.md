# README images

Screenshots referenced by the top-level `README.md`. Capture these from a running
instance (the sandbox seed account has good sample data — `npm run sandbox`).

| File | What to capture |
| --- | --- |
| `treemap.png` | The main dashboard: the treemap heatmap with several holdings, sized by equity and colored by daily change. Toggle to a state with a mix of green/red tiles. |
| `analytics.png` | The `/analytics` page: snapshot history line chart + the equity allocation donut. |

Guidance:
- Grab them at ~1440px wide, then downscale so each is < ~400 KB.
- A short GIF of import → treemap → analytics makes a great LinkedIn post asset;
  save it as `demo.gif` here and it can be embedded too.
